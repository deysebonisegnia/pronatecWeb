Curso Programador Web

Sobre o curso Programador Web

Você aprenderá a planejar, programar e publicar websites com métodos que permitem a inserção e manipulação de conteúdo dinâmico, além de desenvolver interfaces de acordo com os padões web.

HTML

O HTML é a limguagem base para se desenvolver qualquer site. É uma linguagem baseada em marcação,onde marcamos os elementos para definir quais informações a página vai ser exibir.A linguagem HTMl foi criada na década de 1990 e suas especificações são controladas pela W3C(World Wide Web Consortium).

CSS

O CSS é uma linguagem de folha de estilo composta por "camadas",criado com o propósito de estilizar as páginas do HTML, ou seja, definir a aparência das páginas para deixá-las visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, posicionamento dos elementos, layout e muito mais. Uma das maiores vantagens de se utilizar o CSS é que ele efetua a separação entre o formato da página e o conteúdo de um documento, proporcionando assim uma maior flexibilidade e controle de como as características serão exibidas,além de reduzir a duplicação no conteúdo estrutural de uma página.
